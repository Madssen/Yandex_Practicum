| Исследование технологического процесса очистки золота | Спрогнозировать концентрацию золота при проведении процесса очистки золота | Python, Pandas, Matplotlib, Numpy, Scikit-learn, исследовательский анализ данных | анализ данных, регрессия, кастомные метрики |

Строится модель машинного обучения для промышленной компании, разрабатывающая решения для эффективной работы промышленных предприятий. Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды на основе данных с параметрами добычи и очистки. Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками.
